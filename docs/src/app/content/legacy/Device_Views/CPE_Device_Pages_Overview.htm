CPE Devices Overview

You can access CPE devices by MAC Address using the Search function or by clicking a CPE device hyperlink on other pages. For example:

Maps provide pop-up scorecard information for sign-on status and other metrics.

Device List tab in the Node view provides links to devices.

Summary tab in the Account View provides links to devices on the Account.

RIO displays the following CPE device pages:

Cable Modem provides Data services. Cable modem types include the standard data cable modem as well as the ADSG and Hydra cable modems.

MTA (Multi-Terminal Adapter) provides Voice services. The MTA is shown as a child of the cable modem.

Set-top Box (STB) provides Video services. STB types include Cisco Scientific Atlanta (SA), Motorola, Hydra and IP STB connected to the Hydra.
