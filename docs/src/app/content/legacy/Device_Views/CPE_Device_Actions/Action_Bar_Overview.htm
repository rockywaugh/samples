Using the Device Actions Menu

The Cable Modem, Set-top box, and MTA device pages contain an Actions menu in the page header. This menu lists available functions the user can perform on the device for troubleshooting.

Click the Actions drop-down and select an option from the menu.

RIO displays only the actions appropriate for the selected device. The following table lists available actions.

If no options are available for a device, the Actions menu appears grayed out.

Actions Menu Option

Allows you to . . .

Admin Ports

Open the following admin ports: HTTP, Telnet, and SSH.

Note: This action is applicable only for devices that support it.

Clear Flaps

Reset the flap count to zero.

Device Watch

Configure variables and time to monitor the device. If the device currently has an active watch configured, this button reads Device Watch (active).

DNS Redirect

Change the user's options for DNS Redirect, to opt-in or opt-out.

Ping Details

Initiate a ping to the CPEâ€™s IP address.

Quarantine/Unquarantine

Quarantine (disable) a modem or enable a modem that has been quarantined.

Note: The Quarantine action is available only to users with special permissions for this feature

Reset

Reboot the modem or MTA.

Reset eCM

Reset the embedded Cable Modem (eCM) on SA ADSG or Hydrase set-top boxes.

Reset Pin

Reset the PIN for parental controls or purchases on Hydra set-top box.

Restart Browser

Restart the device browser for an IP Set-top box.

Delete Cookies

Delete device cookies for an IP Set-top box.

Clear Storage

Clear the device storage for an IP Set-top box.

Reload HNAV

Restart the HNAV components on an IP Set-top box.

Although not in the Action Menu itself, an additional Action on the Cable Modem pages is the Create a Cable Modem Ticket action.
