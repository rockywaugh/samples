Ping Details

Although RIO displays CM Ping status on the device page to indicate whether the modem is alive, you can view more detailed ping results using the Ping Details action. This action allows you to view additional latency or packet loss details. You can also ping the gateway or any bridged devices.

To ping a device:

Navigate to the applicable modem.

On the device Summary tab, click Ping Details in the Actions Menu. RIO displays the Ping Details dialog box.

Select the Pings, Interval, and Packet size options from the drop-down lists.

Select the Device (options include the cable modem, gateway, or any bridged devices).

Click Ping. RIO displays the results, including summary information.

Close the dialog box when finished.
