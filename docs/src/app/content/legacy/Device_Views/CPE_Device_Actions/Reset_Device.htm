Reset Device

The process to reset devices is basically the same for each device type; however, the Reset cable modem is slightly different.

To reset a device:

Navigate to the applicable device.

On the device Summary tab, click Reset in the Actions Menu. RIO displays the Reset device dialog box.

Click Reset. RIO reboots the device.

To reset a cable modem:

When resetting a cable modem, you can select to reset the device via SNMP or via CMTS. When resetting the modem via CMTS, the cable modem is removed from the periodic ranging list on the CMTS which forces the modem to reset.

Navigate to the applicable device.

On the device Summary tab, click Reset in the Actions Menu. RIO displays the Reset Cable Modem dialog box.

Select whether to reset the modem via SNMP (Reset Device option) or via CMTS.

Click Reset. RIO reboots the device.
