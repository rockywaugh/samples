Reset PIN

The Hydra set-top box provides the option to reset the PIN (4 digit Personal Identification Number) for either Parental Controls or Purchases.

To reset the PIN:

Navigate to the applicable set-top box.

On the Hydra STB tab, click Reset PIN in the Actions Menu. RIO displays the Reset PINs dialog box.

From the Type drop-down list, select the PIN type to reset: Parental Controls or Purchases.

Enter a 4 digit PIN. The default PIN is 0000 (this is also what the PIN resets to if the field is left blank).

Click Reset. RIO resets the PIN to what was entered.

Click Done.
