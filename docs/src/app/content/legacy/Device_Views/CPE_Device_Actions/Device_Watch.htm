Device Watch Action

To add a device to Device Watch:

Navigate to the applicable modem.

On the device Summary tab, click Device Watch in the Actions Menu. RIO displays the Add dialog box.

Select the metric(s) you want to monitor from the drop-down list.

The Default CPE Metrics option includes: HSD Transmit, HSD Receive Power, HSD Signal to Noise, HSD Downstream Freq and HSD Upstream Freq.

If you select options other than the Default CPE Metrics, RIO displays them individually, as shown below.

Select the Start and End Times for monitoring. Use the Calendar icons (as shown below to display the Calendar).

If you want to receive e-mail notifications from the device, enter your e-mail address. You can also add additional e-mail addresses (on separate lines).

Click Add. RIO displays a confirmation message and adds the device to Device Watch.

The watch is viewable on the Device Watch tab in the Device view and on the Division view's Device Watch tab.
