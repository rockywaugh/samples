Create Cable Modem Ticket

To create a trouble ticket form the Cable Modem page:

Click Create a Cable Modem Ticket, in the upper right corner of the Cable Modem pages. RIO Displays the Choose a Survey dialog box.

Select a Survey from the drop-down list.

Complete the required fields on the form. The fields vary depending on the Survey, so be sure to use the scroll bar, if applicable, to view all fields.

Click Save. RIO submits the ticket to a queue for processing.

For additional details and information for submitting the form, see Creating a Trouble Ticket.
