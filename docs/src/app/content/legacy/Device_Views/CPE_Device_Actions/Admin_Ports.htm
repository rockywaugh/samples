Admin Ports

For devices that support it, you can use this action to open the following admin ports: HTTP, Telnet, and SSH.

In addition, RIO provides access to the Management Portal for IPv6 modems with HTTP supported and enabled.

To configure admin ports:

Navigate to the applicable modem.

On the device Summary tab, click Admin Ports in the Actions Menu. RIO displays the Configure Admin Ports dialog box.

Click Enable next to the appropriate option, if it is available for the selected device.

If options are not available, RIO states “Not supported.”

Click X to close the modal window.

Management Portal

A Management Portal login option is available for IPv6 modems where HTTP is supported.

To access the Management Portal:

Navigate to the applicable modem.

On the device Summary tab, click Admin Ports in the Action bar.

Click Enable next to HTTP.
