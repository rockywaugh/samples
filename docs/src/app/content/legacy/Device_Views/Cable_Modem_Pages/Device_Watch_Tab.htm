Device Watch Tab

The Device Watch tab in the Cable Modem view displays the Device Watches that are currently configured for the device (see Device Watch Action for how to configure a Device Watch).

To view only watches you have set up for the device, click My Watches.

Click on a hyperlinked Account Number or MAC Address to jump to the Account or Device Summary page.

For information on using additional sort and search features in this table, see Using Table Tools and Features.

To stop the watch, click Running in the Status column. A sample is shown below.

You can stop only watches that you have configured.

To view additional details, click View in the Results column. A sample is shown below.

The Results pop-up shows the metrics selected for the Device Watch.
