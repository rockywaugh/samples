Spectrum Analyzer Tab

The Spectrum Analyzer tab in the Cable Modem view displays downstream spectrum analysis data for modems that support the analyzer. A sample is shown below.

Select data to view from the options at the top of the graph.

Hover over points on the graph to view pop-up data.

Select how many data sets to graph, from the Graphing drop-down list (1-5, 10, 15, or 20).

By default, RIO displays average and range data for 15 channels; however, you may select more or fewer channels to view. Use the bar at the bottom of the graph to select a channel range, as shown below.

Click the Current Channels links to select a channel; or

Click and drag your mouse across the bar to select the channels you want to view.

To view historical data sets you have previously polled, click Show Historical Data Sets, as shown below.

Select the data sets you want to view and click Graph Selected Data Sets. RIO displays the graphs for the selected sets.

To refresh data, click Poll Device for New Data. RIO displays a New label next to the data set in the Graphed Data Sets list.
