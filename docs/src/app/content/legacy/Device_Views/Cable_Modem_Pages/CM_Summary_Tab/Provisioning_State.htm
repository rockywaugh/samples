Provisioning State

The Provisioning State panel displays provisioning status information and indicates if any differences exist between LeaseLogger/MAC to IP and LDAP.  A sample of a healthy device with no discrepancies (the billing feature codes match what is provisioned in LDAP and what is active on the device) is shown below.

This panel refreshes automatically when the status changes from “offline” to “alive” mode. Hover over items in the panel to view additional information. A sample is shown below.

Hover over each Feature Code to see a definition of the code.

Hover over Provisioned Features to see provisioning information from the billing system and LDAP (a sample is shown above).

Hover over Active Features to view the active feature codes on the device.
