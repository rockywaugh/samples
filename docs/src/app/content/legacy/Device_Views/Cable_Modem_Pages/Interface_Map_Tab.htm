Interface Map Tab

The Cable Modem Interface Map tab provides information about CMTS, Modem, Integrated Cable, and Upstream data. A sample is shown below.

The Interface Map tab contains the following panels. 

Panel

Displays . . .

Mac Domain Interface Map

Description, Status, Power, and Modulation information about the CMTS and the Modem.

Downstreams

Downstream information for the CMTS and modem including frequency, Bandwidth in and out, Power, Noise, and Status.

Upstreams

Upstream information for the CMTS and modem including frequency, Bandwidth in and out, Power, Noise, and Status.

For Downstreams and Upstreams, click the separate upstream and downstream links to view applicable data.
