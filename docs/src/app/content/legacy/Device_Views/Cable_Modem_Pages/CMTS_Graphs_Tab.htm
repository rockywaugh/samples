CMTS Graphs Tab

The CMTS tab in the Cable Modem and the MTA views allows you to view graphs from the Cable Modem Termination System (CMTS) interface for the upstream channels. A sample is shown below (the tabs for both Cable Modem and MTA display the same type of data).

Select a bonded up-stream from the drop-down menu (these may vary).

Click the Jump to Interface link to go to the CMTS Interface view.

Select a tab to view available graphs by type. Available types are listed below.

Click an individual graph to view available timeframe graphs for the selected graph (i.e. hourly, daily, weekly, monthly). Sample timeframe graphs are shown below.
