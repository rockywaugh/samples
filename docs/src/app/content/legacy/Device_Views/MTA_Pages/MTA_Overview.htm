MTA Overview

The MTA page provides information specific to the Multimedia Terminal Adapter (MTA), for Home Phone. If you know the MTA MAC address, use the Search field. However, you can also access the MTA from the Summary tab in the Account View or from the Cable Modem Summary tab.

If the TWC customer subscribes to Home Phone, RIO displays the MTA information in the Component Status panel of the Cable Modem Summary tab (see Child Devices).

To view the MTA Summary, click the MTA MAC Address.

The MTA View contains the following tabs:

MTA Summary

CMTS Graphs

DHCP

Entity History

Family Tree
