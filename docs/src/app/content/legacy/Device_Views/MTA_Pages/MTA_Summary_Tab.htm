MTA Summary Tab

The MTA Summary tab displays information specific to the MTA device, including phone line details and provisioning information. This page also contains an Action bar for performing diagnostic or troubleshooting actions on the modem.

The MTA Summary tab contains the following panels.

Panel

Displays . . .

Summary

CM Init state, CM and MTA ping status, whether the MTA is in quarantine, and battery status. Items in this panel auto-refresh every 5 seconds. Click the Pop-out Window icon to view in a separate window.

LDAP

Information from the LDAP database.

Line Details

Telephone number, real time Hook status, call agent ID, line name, platform, and PIC codes.

Note: Hook status refreshes every five seconds.

Provisioning

MAC address, services platform, IP address, operational status and provisioning information.

Digital Home Phone

Phone service features (e.g. call blocking), voice mail, device ID and type, whether call forwarding is active, and service status.

Phone Line Details

Phone service features (e.g. call blocking, voice mail etc.), line properties, service detais, and termination profiles.

NCS or IMS

Information for either the IMS or NCS platform, including: phone number, data age (last time the device was polled for data) number of lines, BTS, profile and sub IDs, MGW, TSAP address.

Interfaces

Name, status, and type of device interfaces.

For information on available actions, see CPE Device Actions.
