Hydra STB Tab

The Summary or STB tab in the Set-top Box view provides information on the customer’s video services. Depending on the platform and technology of the box, the Summary tab data varies as RIO displays the data as available from the device’s response.

If the set-top box is a Hydra-class STB, RIO displays the Realtime and Health Panels and a list of connected IP set-top boxes.

The Hydra STB tab includes the following panels:

Panel

Displays . . .

Realtime

Real time ping status and transmit and receive levels. RX and TX refresh every 30 seconds.

SPP

Settings, Preferences, and Profiles (SPP) status. The status shows:

Active = The account associated with this box was found in SPP.

Inactive = The account was not found in SPP.

Unknown = The division for the account is not mapped to a billing division.

Health

Health status bar for the device. Select the number of days/weeks you want to view.

Summary

Vendor, Code Revision, Hub, IP Address, Vendor, etc.

Data

Set-top box type, version, model and address information, and customer service information for credit and PPV limits.

Levels

Multi-media over Coax Alliance (MoCA) summary information, QAM levels, Uptime and Online status. Select the  

Packages

Service packages on the account (shown in comma separated format).

IP Set Top Boxes

Connected IP set-top boxes. Click the Device link to view.

HAM Info

HNav Application Manager (HAM) information, including Division Code, Device info, Software and Version info, etc. This information is refreshed every 12 hours.

Status Code may display one of the following, along with a Description:

– REQUESTED_VERSION: The version the device should be running, as provided by HAM. This info is refreshed every 12 hours after initial start.

– PHONED_HOME: The HNav client makes this call upon being loaded onto the device.

– BOOT_INFORM: Boot messages come via the TR-069 and the ACS from the device. A “healthy” box will not be in this state for long, as it should go to PHONED_HOME data once the HNav client starts.

– PERIODIC_INFORM: Provided via the TR-069 from the device and sent every 24 hours.

– NON_RESPONDER: This status is placed on the device if there have been no messages for 10 days.

A “healthy” box normally displays a status of REQUESTED_VERSION, PHONED_HOME, or PERIODIC_INFORM, which is refreshed every 12 hours

VOD Stream Information

Video on Demand (VOD) stream information, including start and end time and Service Group. Select the number of days to view from the drop down list: 1–7, 14, 30, or 60.

VOD Error Information

Video on Demand (VOD) error information, including code, time, and Service Group. Select the number of days to view from the drop down list: 1–7, 14, 30, or 60.
