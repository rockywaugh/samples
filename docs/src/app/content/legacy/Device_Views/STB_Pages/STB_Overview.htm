STB Overview

The Set-top Box (STB) pages provide information specific to the customer's video services. If you know the MAC address, use the Search field to access the device pages. However, you can also access the STB from the Summary tab in the Account View.

Depending on the platform (e.g. Scientific Atlanta or Motorola) and technology (e.g. ADSG or non-ADSG) of the box, the RIO displays varying data as available from the deviceâ€™s response. If RIO is unable to retrieve any data, it displays an error message.

The Set-top box view contains different Summary or STB tabs, depending on the box:

Scientific Atlanta Summary Tab

Motorola Summary Tab

Hydra STB Tab

IP STB Tab

In addition the Set-top Box view may contain the following tabs:

STB Diagnostics Tab

Signal Levels Tab (similar to the Cable Modem Signal Levels Tab).
