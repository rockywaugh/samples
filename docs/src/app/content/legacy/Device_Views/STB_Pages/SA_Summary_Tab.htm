SA Summary Tab

The Summary or STB tab in the Set-top Box view provides information on the customer’s video services. Depending on the platform (e.g. Scientific Atlanta or Motorola) and technology (e.g. ADSG or non-ADSG) of the box, the Summary tab data varies as RIO displays the data as available from the device’s response. If RIO is unable to retrieve any data, it displays an error message.

The Summary tab for set-top boxes on the Scientific Atlanta (SA) platform is similar for both ADSG and non-ADSG set-top boxes. A sample is shown below.

The Summary tab displays the following panels.

Panel

Displays . . .

Realtime

Real time ping status and transmit and receive levels. RX and TX refresh every 30 seconds.

Health

Health status bar for the device. Select the number of days/weeks you want to view.

Summary

Vendor, Code Revision, Hub, IP Address, Vendor, etc.

Data

Set-top box type, version, model and address information, and customer service information for credit and PPV limits.

Levels

Service levels and frequencies for Forward and Receive Data Channels, online status, overall health, QAM levels, etc.

Authorized Packages

Service packages on the account.

VOD Stream Information

Video on Demand (VOD) stream information, including start and end time and Service Group. Select the number of days to view from the drop down list: 1–7, 14, 30, or 60.

VOD Error Information

Video on Demand (VOD) error information, including code, time, and Service Group. Select the number of days to view from the drop down list: 1–7, 14, 30, or 60.
