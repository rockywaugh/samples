Navigating CPE Device Pages

This section provides a summary of page elements for CPE devices and explains how to navigate and use features unique to the device pages. A sample page for a cable modem is shown below.

The CPE pages contain different tabs as applicable for the selected device.

 

Use the Service Path sidebar to navigate to other pages associated with the account or specific device.

Click the Pop-out Window () icon where available to view the selected information in a separate window (see Using_Pop-Out_Windows).

Use the Toolbar and Device Toolbar as you would on any entity page.

Use the Actions Menu to perform functions on the device for troubleshooting.

Viewing Health Bar Status

The Health panel on a device page displays a status bar depicting device health. A sample is shown below.

Enter the number of days or weeks you want to view. Use the buttons to select Days/Weeks.

Hover over the Health bar for additional pop-up information about the time and duration of the status segment. This is particularly useful if the bar displays disruptions (shown in red). A sample is shown below.

Click the Pop-out icon () to monitor Health status in a separate window (see the following section).

Using Pop-Out Windows

A few CPE pages provide pop-out windows for panels providing real time polling updates or graphs. These panels are identified with a Pop-out Window icon (). A sample is shown below.

Click the Pop-out Window icon to open the Window. You can minimize or move the Window to another place on your desktop.

Click the Close button to close the Pop-Out Window.

Click Resume to view status in the main RIO window again.

If you are idle on the same page in RIO for some time, real time polling pauses.

To resume polling, click I'm here!.
