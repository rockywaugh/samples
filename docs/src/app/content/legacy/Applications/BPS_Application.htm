BPS Application

The BPS Application tool allows you to query the Broadband Provisioning System, to view Account Number, Name, User ID etc. as listed in BPS. This is the same functionality as in the NYROC Account Lookup tool.

To query BPS:

From the Menu, select Applications>BPS>[Region].

Select a Site from the drop-down menu (some regions have only one site).

Select a Search Type from the drop-down menu. Account Number is selected for this example.

Enter your Search Criteria.

For example, if you selected Account Number for Search Type, enter an Account Number in the Criteria field.

Click Search. RIO displays the results.

For information on using the sort, filter, and other features in this table, see Using Table Tools and Features.
