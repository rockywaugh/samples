IQD Application

The IQD Application interface allows you to query IQD Streams and Alarms, for up to 25 servers at a time.

The IQD Streams filter allows you to search the number of unique and total views for selected channels (stream source IDs).

IQD alarms filter allows you to view the average alarms per second for a specified alarm type.

To access IQD interface:

From the Menu, select Applications>IQD.

This page lists servers and their status. If the status is red, they are not currently searchable.

Select the IQD Streams or IQD Alarms tab. This example uses IQD Streams (both tabs are similar).

From the Filters, select a Server from the drop-down list, and click the Add button (). RIO displays the selected server and time-frame calendars.

You can add up to three servers to search.

From the Server panel, select the Sources to search.

For IQD Streams, these are channels (as shown below). For Alarms select Alarm Types.

Begin entering a search term (channel name) to view a list.

Make your selection from the list.

Click Add. RIO displays the selection to the right.

Select the From and To time-frames to search.

Select a date from the calendar.

Use the clock arrows to adjust the time of day to search.

For IQD Streams, you can select a time up to 4 hours only.

Click Search. RIO displays the results.

For IQD Alarms, you also have the option to download the search results to a CSV file. Click Download CSV.
