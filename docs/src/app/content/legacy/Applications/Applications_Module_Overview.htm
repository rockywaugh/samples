Applications Module Overview

The Application Module provides interface access to several tools that provide data to RIO. This access can assist users in troubleshooting, as well as provide information on devices that are not yet populating in RIO.

This module includes direct access to the following:

IQD Application to query IQD streams and alarms.

Mail LDAP to query TWC's Mail LDAP database, to view general mailbox and host information.

Modem LDAP to query TWC's Modem LDAP database, to view general modem or MTA information.

BPS Application to query the Broadband Provisioning System (by region), to view Account Number, Name, User ID etc. as listed in BPS.

Xymon Device Groups. This is currently under development and not open yet to users.

To access the tools in this module, select an option from Applications in the Main Menu.
