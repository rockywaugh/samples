Modem LDAP

The Modem LDAP feature allows you to search TWC's Modem Lightweight Directory Access Protocol (LDAP) database, to view general modem or MTA information. You can query the database using either a modem MAC address or a phone number when searching for an MTA.

To query Modem LDAP:

  1. From the Menu, select Applications>LDAP>Modem.

  2. Select the Market from the drop-down list.
  3. Select Search Type (either Cable Modem or Phone Number) from the drop-down list.
  4. Enter the MAC Address or Phone number, depending on Search Type selected.
  5. Click Search. RIO displays the results. A sample for Modem MAC Address search is shown below.
