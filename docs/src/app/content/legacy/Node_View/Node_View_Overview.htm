Node View Overview

The Node View is a tabbed view that displays information specific to a single node. You access this view by clicking any Node hyperlink on other pages (e.g. Family Tree, Service Path, Status Map, etc.), or you can search for a specific node using the Search Function in the Toolbar.

The Node View includes the following tabs:

Scorecards

Device List

IVR

Impact Description

IMS Scorecard

Map

Ping

PNM

Signal Levels

Spectrum Analyzer

Family Tree

Entity History
