Node Scorecards Tab

The Scorecards tab in the Node View is the default landing page when accessing a Node, unless you have changed your View Preferences. The Scorecard panels provide snapshots of network and device information.

The Node Scorecards tab includes the following panels.

Panel

Displays . . .

Sign On

Sign-on information for accounts and components attached to the Node. This panel replaces the CPE Scorecard in other views.

Children by Type

The number of Accounts attached to the Node.

DOCSIS Versions

The number of devices by DOCSIS version on the Node.

ISPs

Internet Service Providers (ISPs) by subscription number.

CPE Scorecard

Network health from the perspective of CPE sign-on level. This panel includes total number of devices or subscribers by type, the number offline, and percentage online.
