PNM Tab

The PNM (Proactive Network Maintenance) tab in the Node view displays the results of signal negotiation between a cable modem and the CMTS to mitigate distortion (this is known as “pre-equalization”). A sample is shown below.

For more information about PNM, see PNM Tab (in the Cable Modem view).

Use the Filters to select Channel and Match Strength.

Select a checkbox to view or hide the group shown on the map, or hide the graph.

You can view a Summarized or Detailed view of the Energy graph, by clicking the appropriate button in the toolbar.

The Summarized Energy view displays dots on the graph (as shown above). Larger dots indicate more customers fall in the specified range.

The Detailed Energy view displays bar graphs. A sample Detailed view is shown below.

To view more detailed device information, click Display PNM Detail. RIO displays the PNM Detail table.

For information on using the sort and search features in this table, see Using Table Tools and Features.

On the PNM tab, use the Map to locate devices based on health. A sample is shown below.

Select Options to filter your results based on device health.

Click the House icon to view pop-up scorecard information.

For additional information about Maps in general, see Map Tab.
