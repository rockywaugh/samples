IMS Scorecard

The IMS Scorecard tab in the Node view lists the number of subscribers with MTAs provisioned on the IMS platform. A sample is shown below.

Click on the hyperlinks to view the Account Provisioning or MTA Summary tabs.

Export the Summary Data (as listed on the IMS Scorecard tab) or MTA Details to an Excel spreadsheet by selecting an option from the Export drop-down list.

Summary Data report provides the information in the IMS Scorecard panel, listing  the number of provisioned and registered MTAs.
