Impact Description Tab

The Impact Description tab in the Node View is similar to Textual Description of Node tab in Unified and displays offline status for devices and areas with accounts that have offline devices, which may be due to an outage or other service interruption.

The Impact Description tab includes the following panels.

Panel

Displays . . .

Connectivity and Service

Associated hub and connected services for the Node.

Affected Accounts

List of accounts with at least one offline device.

Address Summary

The areas (street blocks) with the most affected accounts (accounts with offline devices).

Node Status

 The online/offline status of connected devices and accounts.
