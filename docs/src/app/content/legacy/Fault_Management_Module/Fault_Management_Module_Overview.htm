Fault Management Module Overview

The Fault Management module provides access to the following:

Alarm Management which provides access to the Netcool Alarms  dashboard for viewing Netcool alarm details.

Trouble Ticket Survey Groups which provides forms for creating a ticket in Eventrak.

Unresolved Outages Dashboard which displays ticket information from Eventrak for unresolved outages.

To access the tools in this module, select an option from Fault Management in the Main Menu.
