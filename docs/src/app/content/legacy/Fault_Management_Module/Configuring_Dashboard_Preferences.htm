Configuring Dashboard Preferences

RIO allows you to set your viewing preferences for the Netcool Alarm dashboard, including selecting filters and defining which severities to view, etc.

In addition, you can assign filters to separate pop-out windows. This allows you to customize multiple views of the dashboard.

Configuring Preferences

To set your viewing preferences:

Click the wrench icon (). RIO displays the Set User Preferences modal.

Select any filters (up to 10 filters) from the drop-down list. As you select a filter, RIO lists it to the right of the filter. (See Defining Pop-Out Window Preferences below for information on how to assign filters to different pop-out windows).

Make other selections using the checkboxes or drop-down lists, based on your viewing preferences, as described in the following table.

Item Description

Severities to display

Select which ticket severities to show by checking the corresponding color.

Show absolute time

Changes how the First and Last time columns display time. Absolute vs. relative time.

Choose rows displayed

Select the number of rows to show (e.g. 10, 25, 50, 100 etc.).

Close Modal* after action

Automatically closes the modal windows after you select an action in the modal.

Unselect after closing Modal*

Automatically unselect any alarms you have selected for acknowledgement/ticketing after you close the modal.

Coloring of Alarm Rows

Select to display the entire alarm row or Summary only by severity color.

* Modal = Pop-up dialog boxes that open within RIO (not in a separate browser Window) in which you perform actions within RIO (e.g. Acknowledge Alarms dialog box).

After making your selections, click Save. Then close the modal.

Defining Pop-Out Window Preferences

After selecting your filters, you can assign which filters you want to view in separate pop-windows. This allows you to view multiple consolidated versions of the dashboard at the same time.

To assign filters in separate pop-out windows:

Select your filters from the Filter drop-down list, as described in the preceding procedure.
