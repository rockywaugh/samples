Alarm Management Dashboard

The Alarm Management dashboard provides Netcool alarm details. The dashboard displays a summary of alarm information and provides a link to view additional alarm details. You can also set user preferences for viewing alarms in this dashboard.

The dashboard also allows you to acknowledge or create a ticket for an alarm or view existing ticket details.

To view the dashboard:

From the Menu, select Fault Management>Alarm Management.

Use the View drop-down list to narrow your results. These lists are from Netcool.

Use the Filters drop-down list to show actionable or maintenance alarms. Use the Filter Results text box to further narrow results on the page (see Using Filters).

When you select one or more alarms, RIO displays the Manage Alarms drop-down menu (see Managing Alarms).

If the alarm is already ticketed, RIO displays the Status and a Ticket link. RIO displays links for Trouble, Change, and Task tickets as appropriate. A sample is shown below. For additional information, see Viewing Ticket Details.

 

Using Filters

The Alarm Management dashboard provides a Filter drop-down list and a Filter Results text field to further narrow results.

Use the Filters drop-down list to select which alarms to show.

You can filter by Actionable or Maintenance status, or view All tickets regardless of status.

Actionable alarms are alarms that are either Active with an open Ticket but are not Acknowledged, or are Cleared but not Acknowledged (ticket requirement).

If you select Actionable in the filter, RIO displays only alarms that still have outstanding actions. 

Hold refers to all alarms that are properly associated with a ticket (Active, Acknowledged with Open Ticket or Cleared, Acknowledged at all), but which were just recently associated with a ticket within the past five minutes.

Act + Hold is the combination of Actionable and Hold. With this option, RIO lists currently outstanding issues, plus issues that were just recently associated with an open ticket.  This allows you to reference the alarm for a brief period of time after which it disappears from view.

Delay Actionable allows you to enter a time-frame in minutes to view alarms that have not been acknowledged in X number of minutes.

You can also filter alarms by Maintenance status and view only Maintenance alarms (Only Maint) or to not show any Maintenance alarms (No Maint).

Use the Filter Results text field to further narrow results and Columns drop-down list to customize your view. For additional information on using the sort and filter features in this table, see Using Table Tools and Features.
