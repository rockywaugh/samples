Acknowledging and Ticketing Alarms

RIO allows you to acknowledge or ticket an alarm from the Alarm dashboard.

To act on an alarm:

Select the checkbox next to the corresponding alarm(s) (you can select more than one) you want to acknowledge or ticket. After selecting a checkbox, RIO displays the Manage Alarms drop-down menu, as shown below.

Acknowledging an Alarm

To acknowledge the alarm(s):

Select Acknowledge Alarms from the Manage Alarms menu. RIO displays a confirmation.

Click Acknowledge. RIO displays your name in the Acknowledge column for the alarm(s) on the dashboard.

Ticketing an Alarm

To create or associate an existing ticket to the alarm(s):

Select Ticket Alarms from the Manage Alarms menu. RIO displays the Ticket Alarms page.

If creating a new ticket:

Select Create ticket.

Review and update the selections from the drop-down lists, if necessary.

Click Create Ticket. RIO displays your name in the Acknowledge Column and adds the ticket number to the Ticket column on the dashboard.

If associating an existing ticket:

Select Enter an existing ticket number.

Select the Ticket type (e.g. Trouble, Change, Vendor, etc.) from the drop-down list.

Enter the Ticket number in the text field.

Click Associate Ticket. RIO displays your name in the Acknowledge Column and adds the ticket number to the Ticket column on the dashboard.
