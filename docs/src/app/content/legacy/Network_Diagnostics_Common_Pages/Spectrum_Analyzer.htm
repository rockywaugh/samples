Spectrum Analyzer Tab

RIO provides a Spectrum Analyzer dashboard for the Enterprise level down to the Node level. The Downstream Spectrum Analyzer is built into select DOCSIS 3 cable modems and allows Technical Operations Center (TOC) and Field Engineering personnel to remotely diagnose customer signal levels more accurately.

On this dashboard, RIO displays a list of known cable modem devices that support Spectrum Analyzer.

To view the dashboard, navigate to the entity and select the Spectrum Analyzer tab, or use the Service Path sidebar entity drop-down menu. A sample of the dashboard at the Market level is shown below.

At the Enterprise, Market, and Division levels you can select how to group the results by making a selection from the Group by drop-down list. Click on the linked Market or Division name to view more detailed lists.

If you are viewing a Division, Hub or Node level, RIO displays the Account and MAC Address of the device. A sample is show below.

Click the hyper-linked Hub or Node to view the Spectrum Analyzer dashboard for only that Hub or Node.

Click the hyper-linked Account Number to jump to the Summary tab in the Account View.

Click the hyper-linked MAC Address to jump to the Spectrum Analyzer Graph for the selected modem.
