IVR Tab

RIO provides an IVR Calls dashboard for the Market, Division, Hub, and Node Views. IVR Calls are calls the customer initiated to TWC Customer Care and which were routed through the Interactive Voice Response (IVR) system.

The Account View also includes IVR data specific to the Account.

To view IVR calls, navigate to the entity and select the IVR tab, or use the Service Path entity drop-down menu.

The IVR tab contains two sub-tabs: Summary and Calls.

IVR Summary Tab

The Summary tab on the IVR page displays compiled data related to IVR Calls and provides a listing of Most Calls per Division, Hub, Node, Code, and Account, as applicable to the entity you are viewing. A sample for Market is shown below.

To auto-refresh the page, use the Auto Refresh toggle button. When turned on, RIO auto-refreshes this tab every 5 minutes. RIO displays a count-down timer when Auto-refresh is turned on.

Use the Filter options to select call type and specify a timeframe for calls  (e.g. 30 Minutes or 2 Hours, etc.).

For information on using additional sort and search features in this table, see Using Table Tools and Features.

Click the pop-out icon () to view the Most Calls per Division, Hub, Node information in a separate Window.

IVR Calls Tab

The Calls tab on the IVR page displays a searchable table listing the details of the IVR Calls for the specified timeframe.

Use the Filter options to select call type and specify a timeframe for calls  (e.g. 30 Minutes or 2 Hours, etc.).

For information on using additional sort and search features in this table, see Using Table Tools and Features.

Click Lookup Accounts or Poll Accounts to go to the Bulk Lookup tool. RIO automatically populates the Lookup tool with the Account numbers in the IVR Calls table.
