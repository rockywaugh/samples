Work Orders Tab

RIO provides a Work Orders dashboard for the Division, Hub, and Node Views. This tab displays compiled data related to Trouble Call and Service Request Work Orders as applicable to the entity.

The Account View also includes a Work Order tab specific to the Account.

To view, navigate to the entity and select the Work Orders tab, or use the Service Path entity drop-down menu. A sample of the Division Work Orders tab is shown below.

Use the Filters to select:

Work Order Type: Trouble Calls or Service Requests

Time period: Last XX Hours or Days

Work Order Status: All, Open, or Closed

For information on using the sort, filter, and other navigation tools on the table, see Using Table Tools and Features.

Click the Hub and Node hyperlinks to jump to the respective view.

Click hyper-links for Total or Truck Rolls numbers to view a detailed table of those Work Orders.

Click hyper-links to jump to a specific Hub, Node, or Account view.

The Work Orders tab contains three sub-tab, described below.

Summary Tab

The Summary tab (the default tab, shown above) contains the following panels:

Panel

Displays

Top Open Codes

The top recorded reason codes for opening a Work Order.

Top Close Codes

The top solution codes for closing the Work Order.

Top WO Types

The top recorded Work Order Type.

Most WO: Hubs

The Hubs in the selected Division with the most Work Orders.

Most WO: Nodes

The Nodes in the selected entity with the most Work Orders.

Most WO: Accounts

Accounts in the selected entity with the most Work Orders.

Graphs Tab

The Graphs tab displays a visual graph for Work Order totals by Total or by Top Hubs, Nodes, Open or Close Codes or Type.

Use the Filters at the top of the page, to narrow results.

Select a category to view subsets of Work Orders. A sample for Top Nodes is shown below.

Select options to view or not view a graph from the legend.

Hover over a point in the graph to view additional pop-up details.

For more information about graph features, see Viewing Graphs.
