Scorecards Tab

The Scorecard tab is available in all entity views in the Network Diagnostics module, except for Account and CPE devices. It is the default tab for Enterprise, Market, Division, Hub, and Node views; unless you have changed your View Preferences.

To view Scorecards, navigate to the entity and select the Scorecards tab, or use the Service Path entity drop-down menu. The Scorecard panels provide snapshots of network and device information applicable to the entity level you are viewing. These tabs are similar for all entities, except for the Node Scorecard.

 A sample for Hub is shown below.

The Market level Scorecards display overall CPE and VOD Scorecards for the entity level and scorecards by Division. The Scorecards tab in the CMTS view provides only the CPE Scorecard for devices attached to the CMTS.

The Scorecards tab includes the following panels.

Panel

Displays . . .

CPE Scorecard

Network health from the perspective of CPE sign-on level. This panel includes total number of devices or subscribers by type, the number offline, and percentage online.

VOD Scorecard

Video on Demand (VOD) stream information, including the total number of streams, errors and success percentage for different timeframes.

Children by Type

The number of each entity type considered a child of the current entity.

DOCSIS Versions

The number of devices by DOCSIS version.

ISPs

Internet Service Providers (ISPs) by subscription number.
