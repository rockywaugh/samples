Bulk Impact Tool

When necessary for troubleshooting and to determine how many customers and or devices may be potentially impacted by an outage, you can search multiple Hubs, Nodes, or CMTS' using the Bulk Impact tool.

To access this search tool:

From the Menu, select Search>Bulk Impact Tool.

Enter (or paste) multiple Node, Hub, and/or CMTS IDs in the text box. Separate each entry with one of the following separators: comma (,) space ( ) semicolon (;) or new line.

Click Search. RIO displays the results including a Summary panel and more detailed results in the Found panel. A sample is shown below.

To view or hide the Search panel, click Show Form/Hide Form.

To view or hide the Summary, click Show Summary/Hide Summary.
