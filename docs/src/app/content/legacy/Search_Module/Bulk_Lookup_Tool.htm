Bulk Lookup Tool

When necessary for troubleshooting, you can search for multiple devices or accounts using the Bulk Lookup tool.  This tool allows you to search multiple Account, IP, or MAC addresses.

To access this search tool:

From the Menu, select Search>Bulk Lookup Tool.

Enter (or paste) multiple account numbers, IP addresses, and/or MAC addresses in the text box. Separate each entry with one of the following separators: comma (,) space ( ) semicolon (;) or new line.

Click Search. RIO displays the results including a Summary panel and more detailed results in the Found panel. A sample is shown below.

For searches on account numbers, RIO lists all equipment for the account.

To view or hide the Search panel, click Show Form/Hide Form.

To view or hide the Summary, click Show Summary/Hide Summary.

To view or hide Missed Items, click Show/Hide.
