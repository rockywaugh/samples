Unresolved Outages Dashboard Overview

The Unresolved Outages Dashboard provides ticketing summary information for open tickets and recently closed tickets for planned and unplanned Eventrak outage tickets by Market/Division. RIO pulls this information from Remedy. This is not a real-time dashboard; however, RIO periodically refreshes the data.

To view the Unresolved Outages Dashboard:

From the Menu, select Fault Management>Unresolved Outages Dashboard.

Use the Filter Criteria to narrow results on the Open and Closed Last 24 Hours tabs.

Select the Auto-Refresh checkbox to have RIO refresh the page data.

The Unresolved Outages Dashboard contains the following tabs:

Summary

Open

Closed Last 24 Hours
