Closed Last 24 Hours Tab

The Closed tab on the Unresolved Outages Dashboard displays all tickets updated with a status of Resolved or closed within the last 24 hours.

Use the Filter Criteria to narrow your results by Market/Division.

Select the Auto-Refresh checkbox to have RIO refresh the page data.

You can sort in ascending or descending order for Outage Start Date and Outage End Date, by clicking the column heading.

Use the Search field to narrow results by any text in the table. For example, to find all resolved tickets, enter Resolved in the Search field.

Click the hyper-linked Ticket ID to view details.

For additional information on using Table Navigation tools, see Using Table Tools and Features.
