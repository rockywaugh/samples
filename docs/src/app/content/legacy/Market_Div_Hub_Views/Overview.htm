Market, Division, Hub Overview

The Market, Division, and Hub Views are tabbed views that provide information specific to the selected entity. These views share several common tabs that display information in basically the same layout for each entity; however, some data may vary depending on the level selected.

These views share the following common tabs.

Scorecards

Map

IVR

IMS Scorecard

Spectrum Analyzer

FEC Summary

Family Tree

Entity History

In addition, the following tabs are available in the specified View:

MTA Battery Report (Market view only)

Modem Firmware Report (Market view only)

Tree Map (Hub view only)

Work Orders Tab (Division and Hub views only)
