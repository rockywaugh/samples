FEC Summary Tab

RIO provides a Forward Error Correction (FEC) Summary tab for the Market, Division, and Hub views. The FEC Summary tab shows the amount of data in error that was corrected and uncorrected during the FEC process for each upstream port of a CMTS. This table also shows the number of registered modems on the CMTS and the Signal-to-Noise Ratio (SNR).
