Device Watch Tab

The Device Watch tab in the Division View displays a list of devices with Device Watches running on them.

Click a hyperlinked Account Number or MAC Address to jump to the Account or Device Summary page.

For information on using additional sort and search features in this table, see Using Table Tools and Features.

To stop the watch, click Running in the Status column.

You can stop only watches that you have configured.

To view additional details, click View in the Watch Results column. A sample is shown below.

The Results pop-up shows the metrics selected for the Device Watch.
