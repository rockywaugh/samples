National Entitlements Tab

The National Entitlements tab in the Enterprise view provides an Account lookup to view the services, or channels, a TWC video subscriber should receive on their TWC TV applications, either in home or outside the home (e.g. on a mobile device). A sample is shown below.

To view entitlements:

Enter an Account Number.

Select a Division from the drop-down list.

Click Get Entitlements. RIO displays the form once it has retrieved the entitlements. A sample is shown below.

If the account has a sub-account (more than one user name), you can select a different user from the DSB Username drop-down list. RIO displays information for the Master account by default. Users may have different entitlements on the same account based on parental controls set, etc.

From In Home Status, select In Home or Not in Home depending on where the user is attempting to access content.

If making additional selections, click Refresh Entitlements to update results.

Use the Filter option to narrow results. For example:

Enter a Service Name to easily jump to that service.

To see all items that are blocked, enter X in the Filter. An example is shown below.
