Introduction

In an effort to consolidate tools, Time Warner Cable is developing RIO. RIO is a single-source interface that accesses live data from other sources.

RIO currently contains four modules:

  • Network Diagnostics accesses live data from both Unified and NYROC tools (this contains RIO's original functionality). RIO provides multiple views into the regions without logging into separate tools or opening multiple Windows. In addition, RIO uses the same data thresholds, where applicable, that NYROC and Unified employ.

    RIO displays service-related information in a hierarchical path (from the market level down to the subscriber account and device information) of the critical elements necessary for a service.

    For additional information, see Using the Service Path.

  • Fault Management, which includes:
    • Alarm Management for accessing the Netcool Alarms dashboard.
    • Trouble Ticket Survey Groups for creating a ticket in Eventrak.
    • Unresolved Outages Dashboard for viewing ticket information from Eventrak for unresolved outages.
  • Applications, which includes application modeling for the following:
    • BPS Application to query the Broadband Provisioning System, to view Account Number, Name, User ID etc. as listed in BPS.
    • IQD Application interface allows you to query IQD Streams and Alarms, for up to 25 servers at a time.
    • Mail LDAP allows you to query Mail LDAP using TWC e-mail address.
    • Modem LDAP allows you to easily query modem information by Mac Address or Phone Number.
  • Search. This module contains:
    • Bulk Impact Tool for determining how many customers and or devices may be potentially impacted by an outage. You can search multiple Hubs, Nodes, or CMTS.
    • Bulk Lookup Tool for searching up to 10,000 devices.
