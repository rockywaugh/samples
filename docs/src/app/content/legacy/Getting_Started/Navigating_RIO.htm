Navigating RIO

The Home page contains links for RIO modules and panels as well as other tools and features (e.g. Menu and Toolbar). Other pages in RIO contain additional features (e.g. Service_Path and Additional_Toolbar).

A sample view of RIO's Home page with highlighted features is shown below.

Tech Ops users have a customized home page.

 

Menu

RIO provides a main menu to access the different Modules. To view the menu, click the Menu icon in the Header. A sample is shown below.

Hover over the Module name to view the sub-menu for that module. A sample for Network Diagnostics is shown below.

Click an item in the sub-menu to jump to the selected view.

RIO Toolbar

The RIO Toolbar is available on every page you view. However, the panels vary depending on the page you are viewing.

The Toolbar includes the following functions.

Toolbar Item

Icon

Description

Menu

Access the menu to select Markets or additional modules.

Home

Return to RIO home page.

Search

 

Search for a RIO entity type (e.g. Account Number, Mac Address, Node, Hub, Region, etc.).

Favorites

 

Access a list of bookmarked favorites, or add entities/pages to your favorites list.

History

View a list of recently navigated entities or actions.

Help

Access the RIO Online Help, User Group SharePoint site, and Escalation Chart.

Notifications

 

View any notifications, such as ticketing notifications.

User Menu

Click the drop-down arrow to access User Profile, Device Watch, Themes, and Logout functions.

Home Page Views

RIO engineers are developing different Home Page views based on User Group permissions (e.g. Customer Care, NOC, Tech Ops). This is still under development and the Home Page views currently contain the same panels as the default RIO home page; however, future RIO releases will contain additional updates.

Some users may have access to more than one User Group view. If you are a member of multiple user groups, your Home Page contains a drop-down menu to select a different Home Page. A sample is shown below.

If you have access to more than one User Group View, you can select a default home page.

Modules

RIO contains the following modules, which are continually being expanded and developed.

Module

Description

Network Diagnostics

Provides service-related information in a hierarchical path (from the market level down to the subscriber account and device information) of the critical elements necessary for a service. Accesses live data from both Unified and NYROC tools (this contains RIO's original functionality).

Fault Management

Provides access to view and create Eventrak tickets.

– Alarm Management

– Trouble Ticket Surveys for creating a ticket in Remedy.

– Unresolved Outages Dashboard for viewing ticket information from Remedy for unresolved outages.

Applications

Provides access to:

–  BPS Application to query the Broadband Provisioning System, to view Account Number, Name, User ID etc. as listed in BPS

– Mail LDAP allows you to query Mail LDAP using TWC e-mail address.

– Modem LDAP allows you to easily query modem information by Mac Address or Phone Number.

– IQD Application interface allows you to query IQD Streams and Alarms, for up to 25 servers at a time.

Search

Provides access to

– Bulk Lookup Tool to search for multiple devices or accounts.

– Bulk Impact Tool for determining how many customers and or devices may be potentially impacted by an outage. You can search multiple Hubs, Nodes, or CMTS.
