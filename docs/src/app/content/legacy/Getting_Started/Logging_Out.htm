Logging Out / Session Time-out

You should always log out of the RIO application when you are finished. RIO sessions do not time-out as long as your browser remains open on a RIO session, as RIO uses token authentication (see below).

Logging Out

To exit the application:

Click the User drop-down menu, in the toolbar.

Select Log-Out.

Token Authentication

RIO uses token authentication.  Upon user login, RIO provides a "token" that is good for one hour.  RIO then generates a new token each hour before the previous one expires, as long as your browser window remains open on RIO. 

Closing the browser window stops the generation of new tokens.
