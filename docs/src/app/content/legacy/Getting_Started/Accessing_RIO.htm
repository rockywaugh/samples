Accessing RIO

RIO is a Web-based tool that you access through your computerâ€™s Web browser. RIO is supported in Internet Explorer (IE) 11 and Google Chrome Web browsers.

To access RIO:

In your Web browser, navigate to the following URL: https://rio.twcable.com

The first time you access and log into RIO, RIO displays the following.

Click the appropriate employer. RIO displays the RIO Login page.

If you select the wrong employer, click the I am not a [Company] Employee link to return to the main page.

In the Employee ID field, enter your Employee ID and Password.

RIO login IDs are verified through the selected employers' employee authentication database.

Click Log In. RIO displays the Home page. (For information about navigating the Home page and RIO's features, see Using Application Features and Tools).

After your initial login, the next time you access RIO you will automatically go directly to the Login page, unless you subsequently clear your browser cache. In these instances, RIO will prompt you to select the Employee option again.
