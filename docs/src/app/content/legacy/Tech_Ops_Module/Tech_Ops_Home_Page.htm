Tech Ops Home Page

The Tech Ops home page displays dashboard information for a selected market, including Map, Node, and Alert information. This provides a high level overview of the overall market's health.

This section explains how to select and set a default market and provides information for each panel on this page. A sample of the Tech Ops home page is shown below.

The Tech Ops home page contains the following panels.

Panel

Displays the following for the selected Market

Map

Market level Map. See Map Tab for information on using the Map features and layers.

Node Health

Top 15 worst performing Nodes. See Viewing Tech Ops Graphs.

Alert Overview

Alert numbers by division. See Using the Alert Overview Panel

Division Availability

Division availability metrics. See Viewing Tech Ops Graphs.

Selecting a Market

The information RIO displays in each panel on the Tech Ops home page is based on the market you select to view. For example, if you select the Southeast market, the Node Health panel displays information only for Nodes in the Southeast.

You can select a default market to view each time you access the Tech Ops home page; however, you can also view a different market at any time.

To select a default market:

Click the star () icon. RIO displays the Default Market list.

Select a Market and click Save.

To view a different market at any time:

Make a selection from the Market drop-down list above the Map.

This does not affect your default market selection.

Viewing Tech Ops Graphs

The Node Health and Division Availability panels display data graphs, color-coded by division. On each graph, hover over a data point to view pop-up scores by division for the date selected. A sample of this on the Node Health graph is shown below.

The Node Health graph also contains hyper-links to the nodes listed. Click the link to the jump to the Node Scorecard tab for that node.

Using the Alert Overview Panel

The Alert Overview panel displays an interactive graph showing the number of alerts by color-coded division. You can hover over the pie chart and bar graph to view alert numbers by type and division, as described below.

The default view displays the number of total alarms and percentage by division, at the top, along with a pie chart graph. The bar graphs at the bottom show the total number of alarms by type for the market. A sample is shown below.

Hover over a color in the pie chart to view the number of alarms by type for the division. A sample for Greensboro (green in the pie chart) is shown below.

Hover over a bar graph for an Alert type and the top portion lists the number of alerts for that type by division. A sample for Node DOCSIS Sign On is shown below. This graph shows that Columbia has the most alerts for this type.
