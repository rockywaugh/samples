HFC Alerts Dashboard

The HFC Alerts Dashboard provides information on Node alerts from Netcool. This section provides an overview of the dashboard and features. Use the links provided on this page to view more detailed information on using specified features.

To access this dashboard, select Tech Ops>Alert Dashboard from the Menu. If you are already in the Tech Ops module, select the HFC Alerts tab.

The first time you access the dashboard, RIO displays a Getting Started Wizard to assist you with initially configuring a default filter. You can change your default filter at any time by using the Filter Picker.

The dashboard displays a summary of alarm information and provides links to view additional alarm details and node information. You can also create and share Alert filters, as well as acknowledge an alarm or create a ticket for one or more alarms.

A sample of the dashboard is shown below.

The Total Alerts number shown in the header reflects the number of Alerts based on your filter criteria. As you change filter selections, RIO updates this total.

The following list provides an overview of how to use the dashboard functions.

To refresh the data, click the Refresh icon () in the toolbar. To set an automatic refresh rate, select an option from the drop-down list, as shown below.

Use the Search function and Filters to sort alert types.

Use the Manage Alerts drop-down list to perform actions on selected Alerts, including acknowledging and ticketing alerts.

Click the Alert Key hyperlink to view Alert Details.

The dashboard also displays several key symbols next to the Alert Key so you can easily identify severity, whether the alert is acknowledged or not and whether the alert is actionable. These symbols also provide access to other actions and data views (see Alert Severity Status and Action Symbols).

Click the Ticket number hyperlink in the Ticket ID field to view Ticket details, if the alert is assigned to a ticket.

Use the Health hyperlink to view additional Node health and metrics information.
