Getting Started Wizards

The first time you access the Tech Ops HFC Alerts Dashboard or the Ticket Dashboard, RIO displays a Getting Started wizard so you can set initial default filters. This section walks through each wizard.

A sample page is shown below.

HFC Alerts Dashboard Wizard

To complete the wizard:

On the initial Getting Started page, click Next to proceed. RIO displays the Default Filter page.

Select an Alarm from the list by clicking in the row.

Click Next. RIO displays a confirmation page.

Click Done. RIO displays the HFC Alerts Dashboard.

You can change your default filter at any time by using the Filter Picker.

Ticket Dashboard Wizard

To complete the wizard:

On the initial Getting Started page, click Next to proceed. RIO displays the Default Ticket Filters page.

Use the Region and Division drop-down lists to select your default location.

Click Next. RIO displays a confirmation page.

Click Done. RIO displays the Ticket Dashboard.
