Viewing Node Health and Metrics

The HFC Alerts dashboard displays a Health hyperlink which provides a score for the overall Node health.

Hover over the Health score to view pop-up metrics for the Node, as shown below.

In the Metrics pop-up, RIO displays the highest (most impacting) value in red text.

Click the Health score to view the Node Health Statistics, as shown below.

In the Statistics modal, RIO highlights the highest (most impacting) value. This is the same value that appears in red text on the Metrics pop-up.

To view additional information, click the HFC Node link to go to the Node Scorecards tab in the Node view.
