Using Alerts Dashboard Filters and Search Functions

The HFC Alerts Dashboard contains a Search function and several Filter options. These options appear in the header.

For information on using the Filter Results field and common table sort and navigation features, see Using Table Tools and Features.

The Search and Filter options unique to the HFC Alerts dashboard are defined below.

Click Search to search and display results by Entity type(s). When using the Search option, you can also create a custom Filter based on your search criteria.

Use the Filter Picker to select, edit, or share a filter. The Filter Picker also allows you set a default filter.

Use the Date Range filter to select a time range for results.

Use the Alert Type drop-down list to sort one of the following Alert types: Node Sign On, Node Power Supply, or VIP.

The filter automatically saves your last selection, so that the next time you return to the dashboard, RIO displays the same selection automatically.

VIP refers to subscribers that are identified as VIP customers as designated in the billing system.

Use the Refine Alerts drop-down list to further filter results by severity, acknowledgement, or maintenance.

Using the Search Function

The Search function allows you to search for Alerts by Entity Type(s) (e.g. division, hub, node, etc.). You also have the option to save your search criteria to create a Filter for future use.

To search Entities:

Click Search in the header. RIO displays the Select Entities drop-down list.

Click Select Entities. RIO displays a Search field.

Enter an Entity name. RIO displays matching entries. Select the entity you want from the list.

You may enter and select more than one Entity.

Click Apply to view Alert results matching your criteria.

Creating and Editing Filters

This section explains how to create and edit a custom filter for specified entities as well as how to use the Filter Picker to select a previously defined and shared filter.

Creating a Filter

To create a filter:

Follow the procedure for using the Search function, as described above (through step 3).

After selecting your entities, click Save as Filter. RIO displays the Create Filter modal.

Enter a Name and Description.

Click Create. RIO saves the filter.

You can use the Filter Picker as described below, to edit, delete, or share the filter, or to set it as your default filter.

Using the Filter Picker

The Filter Picker allows you to select a previously defined filter for viewing alerts. The Picker also allows you to assign a different default filter, share a filter, and edit or delete a filter that you created.

To open the Filter Picker, click Filters and then the Current Filter button in the header. RIO displays the Filter Picker.

The Default filter is identified by the selected radio button next to the Filter.

To select a different default Filter for each time you use the dashboard, click Default next to the Filter Name.

To use a different filter for this session, select a Filter by highlighting it click the Close button ().

To share a filter with other users, select a Filter from the list and click the Share icon (). RIO displays a pop-up with the URL to the filter. A sample is shown below.

You can copy the link and e-mail it to other users.

Editing and Deleting Filters

From the Filter Picker, you can also edit or delete filters that you created.

Select your filter from the Picker, as shown below.

To delete the filter, click the Delete icon (). RIO removes the filter from the list.

To edit the filter:

Click the Edit icon (). RIO displays the Edit Filter modal.

Complete the information on the form and use the Select Entities function as you would when creating a filter.

Click Edit to save your changes.

Refining Alerts

Use the Refine Alerts filter to further narrow your Search or Filter results. A sample is shown below.

Use the Show/Hide buttons to select whether to view Acknowledged and Maintenance alerts.

Hide is the default selection. With this selected, RIO does not display any alerts that are Acknowledged or related to Maintenance.

If you select Show, RIO displays only alerts that are Acknowledged and/or related to Maintenance, it does not display All alerts. A future release will include the option to show ALL alarms for either category.

Use the Severity checkboxes to select which severity levels you want to view.

As you select or change filters, the Total Alerts number changes to reflect the current filter selections.
