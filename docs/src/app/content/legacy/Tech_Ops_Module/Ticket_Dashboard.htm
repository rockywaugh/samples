Ticket Dashboard

The Ticket Dashboard in the Tech Ops Module displays Trouble or Change ticket information from Eventrak.

The first time you access the dashboard, RIO displays a Getting Started Wizard to assist you with initially configuring a default filter.

To access this dashboard:

From the Menu, select Tech Ops>Alert Dashboard.

Select the Ticket Dashboard tab.

To view a dashboard, select the Trouble or Change tab. A sample for Trouble tickets is shown below.

Both Trouble and Change views are similar, with the exception of the Assignee field. On the Change tab, Assignee is replaced with Coordinator.

By default, the Trouble Ticket dashboard displays tickets assigned to you. However, you can delete your name from the Assignee field to view all tickets. Alternatively, you can enter the name of another Assigned user to filter tickets by assignee.

A future release will leave the Assignee field blank and will not default to the current user.

To further filter results on either dashboard, select parameters from the filter drop-down lists (such as Region, Status, Severity, or CTI). Some filters are dependent on the previous variable selected. For example, to select a Division you must first select a Region.

Use the Filter Results field to further narrow results and the Columns drop-down list to customize your view. For information on using these and additional sort and filter features in this table, see Using Table Tools and Features.

Use the Download CSV button to export the table to an Excel spreadsheet.

Click the Ticket number hyperlink in the Ticket ID field to view Ticket details.
