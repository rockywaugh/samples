Viewing Alert Details

In addition to providing Alert summary information in the table, you can access an Alert Details page to view additional information,  by clicking the Alert Key.

The Alert Dashboard also displays several key symbols next to the Alert Key so you can easily identify severity, whether the alert is acknowledged or not and whether the alert is actionable.

These symbols also provide access to other actions and data views as described below.  

Alert Severity, Status, and Action Symbols

The Alerts dashboard provides a color-coded severity indicator plus status and action symbols in the left columns, as described in the following table.

Severity is color-coded and allows you to easy identify if an alert is critical (Red), clear (Green) or other severity.

You can also view Severity in the Alert Details.

You can also sort by severity using the Refine Alerts filter.
