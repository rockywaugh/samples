Viewing Ticket Details

You can view Ticket details from both the Ticket dashboard and the HFC Alerts dashboard, if the alert is ticketed.

RIO displays the hyper-linked Ticket number in the Ticket ID column on each dashboard. A sample for the HFC Alerts dashboard is shown below.

On the Ticket dashboard, RIO displays the Ticket ID

Click the Trouble ID hyperlink to view the Trouble Ticket information for the Alert. A partial sample from the Ticket dashboard is shown below.

To access the ticket in Eventrak, click View in Eventrak. RIO displays the Eventrak log in page.
