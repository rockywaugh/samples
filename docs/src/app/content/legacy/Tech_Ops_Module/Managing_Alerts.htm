Managing and Ticketing Alerts

RIO allows you to acknowledge or ticket an alert from the HFC Alerts Dashboard. In addition, you can also enable/disable maintenance to suppress alerts for a defined period of time.

To act on an alarm:

1.  Select the checkbox next to the corresponding alert(s) (you can select more than one) you want to manage. After selecting a checkbox, the Manage Alerts drop-down list becomes available.

2.  Select the appropriate action from the Manage Alerts drop-down list. Additional information for each action is provided in the following sections.

Acknowledging Alerts

To acknowledge an Alert:

Select the alerts(s).

From the Manage Alerts drop-down list, select Acknowledge Alert.

RIO displays a green check next to the Alert and displays Acknowledgement information in the Alert details, as shown below.

 

To remove an acknowledgement, select the Alert and from the Manage Alerts drop-down list, select Unacknowledge Alert.

Enabling Maintenance

To acknowledge an Alert:

1.  Select the alerts(s).

2.  From the Manage Alerts drop-down list, select Enable Maintenance. RIO displays the Maintenance Alerts modal.

3.  Enter the number of hours you want to suppress alarms and click Enable Maintenance.

RIO displays a green check next to the Alert and displays Maintenance information in the Alert details, as shown below.

To disable the Maintenance suppression of alerts, select the Alert and from the Manage Alerts drop-down list, select Disable Maintenance.

Ticketing an Alert

You can link an alert to an existing Trouble ticket, create a new Trouble ticket for an Alert, or unlink all Trouble tickets.

To ticket an alert:

Select the alerts(s).

From the Manage Alerts drop-down list, select Ticket Alerts. RIO displays the Ticket Alert modal.

Select which option you want to perform. Depending on your selection, RIO displays different options.   

To link to an existing ticket:

Select Link this alert to an existing Trouble ticket. RIO displays any existing Change or Trouble tickets for the node.

Select a ticket from the list, or enter a Ticket Number to search for a ticket.

You can also use the Category drop-down to display only Change or Trouble tickets.

Click Link Ticket. RIO links the Trouble ticket and displays the Trouble Ticket ID in the Trouble ID column for the Alert.

To create a new ticket:

Select Create and link this alert to a new Trouble ticket. RIO displays a ticket form.

If you are ticketing more than one alert, select which alert to use as a template to populate the ticket fields, as shown below.

Complete the form, entering required location and CTI information following your normal ticketing procedures.

Click Create Ticket.

RIO displays a confirmation message with the Ticket Number. RIO also displays the Ticket Number in the Ticket ID column for the alert (see Viewing Ticket Details).

Unlinking a Ticket

To disassociate one or more tickets from an alert:

1.  Select the alerts(s).

2.  From the Manage Alerts drop-down list, select Ticket Alerts.

3.  Select Unlink any Trouble ticket associated with this alert.

4.  Click Unlink Tickets. RIO removes the associated tickets from the alerts.
