Tech Ops Overview

Users with Tech Ops permissions have access to the Tech Ops module within RIO and see a customized home page when logging into RIO. A partial sample of the home page is shown below.

For additional information on the panels and selecting a market on this page, refer to the Tech Ops Home Page.

In addition, Tech Ops users can access several dashboards in the Tech Ops module:

Alerts Dashboard to view alerts from Netcool.

Ticket Dashboard to view Trouble or Change tickets.

Node Health Dashboard to view information on the strength of the Node.
