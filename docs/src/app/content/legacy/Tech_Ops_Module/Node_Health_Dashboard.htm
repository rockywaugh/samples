Node Health Dashboard

The Node Health Dashboard in the Tech Ops Module displays the top 100 worst nodes across the enterprise.

To access this dashboard:

From the main Menu, select Tech Ops>Node Health Dashboard; or

From within the Tech Ops module, select the Node Health Dashboard tab.

Use the Filter Results field to narrow results. For example, you can enter a specific Hub or Node. An example is shown below.
