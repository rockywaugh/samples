IVR Tab

The IVR Calls tab in the Billing Account View displays a list of calls that the customer initiated to TWC Customer Care and which were routed through the Interactive Voice Response (IVR) system.

This tab is also available at the Market, Division, Hub, and Node Views (see IVR Tab).

Use the Filter options to select call type and specify a timeframe for calls  (e.g. 30 Minutes or 2 Hours, etc.).

Select Include Unspecified to include calls not categorized by one of the filter types.

For additional information on using the sort and search features in this table, see Using Table Tools and Features.
