Work Orders Tab

The Work Orders tab in the Account view displays details for Trouble Call and Service Request Work Orders for the account.

Use the Filters to select:

Work Order Type: Trouble Calls or Service Requests

Time period: Last XX Hours or Days

Work Order Status: All, Open, or Closed

With the Truck Rolls tab removed from the Account View, you can check the code in the Order Class column to see if the Work Order is a truck roll. The Order Class column displays a T or TN for truck rolls. If this column does not appear in your view, use the Show/Hide Columns button to view.

For information on using the sort, filter, and other navigation tools on the table, see Using Table Tools and Features.
