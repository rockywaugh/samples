Status Tab (Xymon Info)

The Status tab is the default landing page when accessing a CMTS or other Host, unless you have changed your View Preferences.

RIO displays Event information from the Xymon tool for viewing the health status of a CMTS or other host (as listed in NYROC). This page provides a summary view of tests available for groups of equipment for the selected host and the current status of the test.

To view the output of a specific test, click a Service name.

You can sort the available Service tests by Service Name or Color in Ascending or Descending order, by clicking the Sort By buttons.
