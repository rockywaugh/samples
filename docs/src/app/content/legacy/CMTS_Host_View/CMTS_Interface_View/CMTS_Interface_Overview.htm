CMTS Interface Overview

The CMTS Interface View is a tabbed view that displays information for a specific CMTS Interface. The views are similar for all Interface ports (e.g. upstream, downstream, Ethernet, etc.)

You can access this view by clicking any CMTS Interface hyperlink on other pages (e.g. Interfaces tab in the CMTS view) or you can search for a specific CMTS Interface using the Search function.

The CMTS Interface View contains the following tabs:

Summary

Metrics

Statistics

Family Tree
