CMTS and Host Overview

The CMTS and Host Views are tabbed views that displays information specific to a single CMTS or Host. You can access this view by clicking a CMTS/Host on other pages (e.g. Family Tree, Service Path, Status Map etc.), or you can search for a specific CMTS/Host using the Search function in the Toolbar.

These views share the following common tabs:

Status

Alarms

Circuits

CMDB

Interfaces

Tickets

Family Tree

Entity History

In addition, the following tabs are available only in the CMTS view:

RF Interfaces

Scorecard

Spectrum Analyzer
