Changing Page Viewing Preferences

You can change the page viewing preferences for each entity level, by selecting which tabs you want to view and selecting a different tab as your default landing page.

To change preferences:

Navigate to a page on the entity level you want to update. For example, to update Node viewing preferences, navigate to a specific Node.

Click the Preference tool icon (gear icon) to display a list of tabs for the selected entity.

Use the arrows and buttons next to each tab name to make your selections.

Click Set as Default for the selected tab to be the default landing page for the entity.

Click the eye icon () to add/remove the tab from your view.

Use the up/down arrows to change the order of the tabs.

Click Save.

To restore tabs to the RIO default settings, click Restore View Defaults.
