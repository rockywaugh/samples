Accessing Entity Types

To access and view entities, RIO provides two main navigation types:

Direct Navigation takes the user directly to the entity within RIO without visiting any intermediary entity. Use any of the following functions for direct navigation:

Search Function

Favorites (book-marked links)

Recent History list

Relational Navigation provides access to an entity from another Entity View or RIO page via a hyperlink to the entity. Use any of the following functions for relational navigation:

Service Path

Map

Family Tree
