Setting Default Home Page

RIO engineers are developing different Home Page views based on User Group permissions (e.g. Customer Care, NOC, Tech Ops). If you are a member of multiple user groups, your Home Page contains a drop-down menu to select a different Home Page. A sample is shown below.

If you have access to more than one User Group View, you can select a default home page. Once selected, RIO displays this page each time you log into RIO; however, you can still select a different home page as necessary by using the drop-down list.

To select a default Home page:

Click the Preference tool icon (gear icon) to display a list of available views based on your user permissions.

Select an option from the list.

Click Save.
