Using the Search Function

The Search field is available on all RIO pages. The Search field allows you to search on any RIO entity type (e.g. Account Number, Mac Address, Node, Hub, Region, etc.) as well as the following attributes:

– Account Number
  (Billing or Salesforce Account Number)

– Account Telephone Number

– Account E-mail Address

– Digital Home Phone Number

– IPv6 Address

– Street Address

– Trouble Ticket Number. Must enter TRB and then the number.
   For example: TRB383522.

RIO receives device information for Node, Hub, customer accounts, and customer equipment from the billing system feed. This device information may take up to 24 hours to appear in RIO.

In addition to this Search function, RIO provides a Bulk Lookup Tool.

Using the Search Field

To use the Search field:

Enter a search attribute (e.g. Account Number or MAC address) in the Search field.

You must enter at least four characters or select a filter type. If you enter fewer than four characters, RIO prompts you to select a Filter, as shown below.

Click the Search icon ()or press [Enter]. RIO displays the result.

RIO may return more than one result (see Sorting Multiple Search Results below).

To clear the Search field, click the X.

To view additional information and options for Search, click the Help . . . link.

Using the Search Filter

RIO provides a Search Filter, which narrows the data RIO searches, to assist in limiting results and to return results faster. The filter populates automatically as you begin typing in the Search field.  A sample is shown below.

To use the filter, click one of the categories. If you enter more than four characters you can ignore the filter and click the Search icon (); however RIO may return multiple results for different entity types.

To view additional information and options for Search, click the Help . . . link.
