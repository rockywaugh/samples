Using Tabs in RIO

Many views in RIO contain several tabs (pages) which provide more information for the entity you are viewing. Tabbed views in the Network Diagnostics module include Market, Division, Hub, CMTS, Node, Account, and Devices. A sample view with tabs is shown below.

Although tabs vary by entity, several tab features are consistent across many entity views.

You can modify your tab viewing preferences by entity type (see Changing Page Viewing Preferences).
