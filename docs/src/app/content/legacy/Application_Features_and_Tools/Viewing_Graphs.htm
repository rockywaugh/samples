Viewing Graphs

RIO provides several different Dynamic graphs. Some graphs provide trend lines with data points. You can hover over a data point on the graph to view additional pop-up information. A sample is shown below.

In this example, the graph displays information by division and is color-coded so lines correlate to the information in the pop-up.

Some graphs also contain hyper-links to entities. Click the link to jump to the entity view. In the example above, click the Node Name to jump to the Node view.

Other graphs allow you to select data variables and view pop-up information.

Select data to view from the options at the top or bottom of the graph.

Hover over points on the graph to view pop-up data.

Click the Context Menu icon () to view print and download options.

Several graphs are also available in pop-out windows on entity pages. To access, click the Pop-out Window icon (). A sample is shown below.

Several fluctuation graphs contain a Legend for the different graph lines. You can customize which lines to view by clicking the items in the legend to select or de-select the graph. A sample is shown below.

A few real-time graphs also contain a pause/play button to stop or resume the graphing.
