Viewing Current Known Issues

The RIO Home page contains a panel listing current known issues or bugs. Users who find a bug or issue within RIO can check this list to see if the issue is already logged, prior to notifying RIO development. In addition you can use the Search field to locate an Issue item.

This Current Issues panel appears below the Favorites and Recent History panels. A sample is shown below.

The issues list contains information from the JIRA development ticket, including the JIRA ID number and a summary of the RIO bug/issue.

If the issue has an open Remedy ticket associated with it, the title may include the ticket number.

Click the ID Number hyper-link to view Issue Details.

Click More... to view Pending and Closed Issues.

Viewing Issue Details

To view issue details, click the JIRA ID number. RIO displays a pop-up dialog box with additional information. A sample is shown below.

Details provide additional information about the issue, and if applicable the fix release version number.

If necessary, you can enter a comment about the issue. Click Add Comment.

Developers can block comments if necessary.

Viewing Pending and Recently Closed Issues

Click More... in the lower right corner of the panel to view Pending and Recently Closed Issues. A sample is shown below.

Click the ID Number hyper-link to view Issue Details.

Click the Ticket Number to view additional ticket details.

For information on using the sort and search features on this tab, see Using Table Tools and Features.

Searching for Current Issues

In addition to viewing the Current Issues panel, you can use the Search field to locate an issue or determine if an issue has been logged.

You can search on the RB (RIO Bug) ID number, if known, or enter a keyword.

Enter the RB number or keywords in the Search Field.

Click Known Issue in the Search Prompt list to narrow the search parameters. RIO displays the search results.

Click the JIRA ID number in the Name column to view Issue details.
