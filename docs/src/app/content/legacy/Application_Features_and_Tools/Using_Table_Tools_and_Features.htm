Using Table Tools and Features

RIO displays data tables on many different pages. These tables provide common features and tools to assist in viewing and sorting the data. This section provides an overview for using the common table tools. A sample table is shown below.

Use the Show Rows drop-down list to select the number of items to view.

Use the Page Controls to move forward or backward through multiple pages of data. You can also select a specific page number to jump to that page.

Sorting Columns

Use the Up/Down arrows () to sort column data in ascending or descending order.

A single up or down arrow () indicates which column is currently sorted.

Using the Filter

Many tables contain a Filter () or Filter Results field. Some filters are free-form text fields where you can enter any data variable from the table on which to filter. Other filters also contain a drop-down list and a free-form text field. The drop-down list allows you to first select a column heading as a variable. An example with both is shown below.

To use this Filter:

Select a Column name from the Filter drop-down list to sort on. (In this example, Model is selected).

You can further narrow your results by then entering a search item related to the selected column, in the text field. (In this example, the user enters Cisco for the Model type to search).

RIO displays only the narrowed results. (In this case, RIO lists all Cisco Models).

 

Show/Hide Columns

Several tables provide a Show/Hide Columns button. This option allows you to select which columns you want to view.

From the drop-down list, select the columns you want to hide or view. Column names that appear in gray are currently hidden.

Changing Date Ranges

Some tables contain an option to adjust the date range for the data shown. An example is shown below.

Click the plus/minus (+/-) signs to add or remove days to the range shown.

Click plus (+) to increase the range in 7 day increments.

Click minus (-) to decrease the range in 7 day increments, if viewing more than 7 days. If viewing 7 days, you can decrease the range in 1 day increments down to the last 2 days.
