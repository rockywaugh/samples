Using Links and Pop-Ups

In RIO, many entities are hyperlinked to their default landing pages. These links allow you to quickly move between pages and view detailed information for entities or devices.

If you are in the Family Tree, clicking a linked entity takes you to the Family tree for that entity.

For example:

On the Account Summary page, click a linked device MAC Address to view the device page.

If viewing a Node map, clicking on an Account Number takes you to the Account Summary.

This also shows an example of a pop-up scorecard.

Clicking an entity in the Service Path also takes you to the default landing page for that entity.

Many items in RIO also provide pop-up status/scorecard or other information or descriptions when you hover over or click the item.

For example:

Click on an entity in a Map view to view pop-up scorecard information (a sample is shown in the previous graphic).
